# Gallery Image Selection - Client Review Process

## מה יש כאן?

תהליך לסימון תמונות של ישובים/כפרים מתוך גלריה של ~12,000 תמונות.

## סטטוס נוכחי

| מדד | ערך |
|-----|-----|
| סה"כ תמונות בגלריה | 11,724 |
| נסקרו עד כה | ~900 |
| אושרו (ישוב/כפר) | 121 |
| נדחו (נוף בלי ישוב) | 43 |
| אחוז אישור | 74% |

## הערכה

- **אומדן שמרני:** ~370 תמונות טובות
- **אומדן אופטימי:** ~500 תמונות טובות

## איך להמשיך?

### 1. פתח את דף הסקירה
```bash
open /Users/aviz/images-finder/gallery/client-review/review_gallery.html
```

### 2. סמן תמונות
- **לחיצה ראשונה** = ירוק (ישוב/כפר - טוב!)
- **לחיצה שנייה** = אדום (נוף בלי בתים / אנשים / לא מתאים)
- **לחיצה שלישית** = ריק (לא בטוח)

### 3. העתק את ה-IDs
בתחתית הדף יש שתי תיבות:
- **GREEN IDs** - התמונות הטובות
- **RED IDs** - התמונות לדחייה

### 4. שלח לאביץ
שלח את שתי הרשימות (ירוק + אדום) כדי להמשיך את התהליך.

## קבצים חשובים

| קובץ | תיאור |
|------|-------|
| `approved_ids.txt` | רשימת IDs שכבר אושרו (121) |
| `rejected_ids.txt` | רשימת IDs שנדחו (43) |
| `review_gallery.html` | דף סקירה עם כל התמונות |
| `data/embeddings.db` | מסד נתונים של ה-embeddings |
| `data/classification.db` | תוצאות הסיווג האוטומטי |

## מה זה "ישוב/כפר"?

✅ **כן:**
- בתים על גבעה
- נוף עם מבנים
- מבט על שכונה/כפר
- גגות ובניינים נמוכים

❌ **לא:**
- נוף בלבד (ללא בתים)
- תמונות של אנשים
- בניינים רבי קומות
- תקריב של פנים
